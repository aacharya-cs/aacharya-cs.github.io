Latex Help



  • latexdiff installation:
    1. download latexdiff from: http://www.ctan.org/tex-archive/support/latexdiff/
    2. go to local directory where latexdiff is downloaded.
    3. type 'make install' from command line by logging in as root('sudo su')--files get stored inside /usr/local/bin and usr/local/man/man1 So have these directories created and 'chmod 777 *'ed.
    4. type 'latexdiff file1.tex file2.tex > diff.tex' from within /usr/local/bin
    5. For tutorials do visit:
      1. http://www.pppl.gov/~hammett/comp/tex/latexdiff.html
      2. http://mirrors.ibiblio.org/pub/mirrors/CTAN/support/latexdiff/README
      3. http://www.tex.ac.uk/CTAN/support/latexdiff/latexrevise
