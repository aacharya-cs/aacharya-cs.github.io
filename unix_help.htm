Unix Help



  • Screen Command in Unix

    • screen -r: shows all screen sessions
    • screen -r screen_number: opens specific screen session
    • ctrl+A+D: Leaves the screen session
    • screen -X quit: terminates all the screen sessions

  • Setting Up New Computer

  • installing openssh server for remote access:
  • sudo apt-get install openssh-server
  • installing java in ubuntu:
  • sudo apt-get update
  • java -version
  • sudo apt-get install default-jre
  • sudo apt-get install default-jdk
  • install oracle java 7 in ubuntu: sudo add-apt-repository ppa:webupd8team/java sudo apt-get update sudo apt-get install oracle-java7-installer
  • install git: sudo apt-get update sudo apt-get install git
  • sudo apt-get install screen
  • install python: sudo apt-get install build-essential sudo apt-get install libreadline-gplv2-dev libncursesw5-dev libssl-dev libsqlite3-dev tk-dev libgdbm-dev libc6-dev libbz2-dev wget http://python.org/ftp/python/2.7.8/Python-2.7.8.tgz tar -xvf Python-2.7.8.tgz cd Python-2.7.8 ./configure make sudo make install
  • pip and easy-install and other python packages: sudo apt-get install python-pip python-dev build-essential sudo pip install --upgrade pip sudo pip install --upgrade virtualenv sudo pip install numpy scipy sudo apt-get install libopencv-dev python-opencv sudo pip install pandas sudo pip install patsy sudo pip install statsmodels pyyaml nltk pymongo
  • installing R: sudo apt-get update sudo apt-get install r-base sudo apt-get install r-base-dev
  • installing GSL: wget http://mirror.sdunix.com/gnu/gsl/gsl-latest.tar.gz tar -xvf gsl-latest.tar.gz cd gsl-1.16 ./configure --disable-shared sudo make make check sudo make install
  • installing Boost: sudo apt-get install libboost-all-dev installing latex: sudo apt-get update sudo apt-get install texlive-full
  • installing subversion: sudo apt-get install subversion
